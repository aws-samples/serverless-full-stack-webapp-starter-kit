# Infrastructure as Code (AWS CDK)

This is the IaC project written in AWS Cloud Development Kit (CDK).

## Useful commands

- `pnpm exec cdk deploy`: deploy the infrastructure
- `pnpm exec cdk deploy --require-approval never`: deploy without confirmation (useful for automation)
- `pnpm exec cdk deploy --hotswap`: deploy with hotswap feature enabled (useful for development)
- `pnpm exec cdk watch`: watch your code and deploy every time changes are detected
- `pnpm exec cdk destroy`: delete the infrastructure
